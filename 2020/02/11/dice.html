Turn images to dice with python

Summary

I received a call from a family member asking me if I could recreate this thing they saw reddit where a guy used 20,000 dice to make an image. Heres the link. I knew someone had to have done this with code and I stumbled upon a medium post post where Nathan Sheppard converted an image into dice. While it was a good start I needed to see how an image would look like with a given amount of dice. Also I wanted to add the ability to add different shades of dice and make it really easy to change these settings.

The first problem was really interesting because it involved a bit of algebra to figure out. Heres the code.
